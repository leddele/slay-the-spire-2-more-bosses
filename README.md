这是一个为《杀戮尖塔 2》(Slay the Spire 2)开发的模组，旨在将部分 Boss 移植到二代中（会有一定修改），以及添加新的bosses。
## 核心功能
 **全3层boss**:都有大量加强
* **腐化之心 (Corrupt Heart)**：作为三层可选 Boss 出现，还原了1代“坚不可摧”和“死亡律动”机制，并有少量强化。
* **建筑师? (The Architect)**：三层高难 Boss，拥有多阶段召唤机器人、骑士和构造体的逻辑。

## 安装方法
1. 确保你已经安装了 [BaseLib] 基础模组。
2. 前往 [Releases](https://github.com/leddele/slay-the-spire-2-more-bosses/releases) 页面下载最新的压缩包。
3. 解压后将整个 `Sts1Content` 文件夹放入游戏的 `mods` 目录下。
   * 路径示例：`Slay the Spire 2/mods/Sts1Content/`

## 包含文件
* `Sts1Content.dll`：核心逻辑代码。
* `Sts1Content.pck`：图片、音效及视觉场景资源。
* `mod_manifest.json`：模组配置文件。

## 开发说明
本模组基于 Godot 4.5.1 (.NET 版) 开发，代码部分使用 C# 编写。
